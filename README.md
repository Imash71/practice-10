# Practice 10 - Game Development (Python Turtle)
## 🎮 Description
This project contains three simple games developed using Python Turtle graphics and console logic:
1. Racer Game
2. Snake Game
3. Paint Application
The purpose of this practice is to improve programming skills, learn game logic, and work with user input and simple graphics.

## 🏁 1. Racer Game
- Control a car using arrow keys (or input)
- Collect coins randomly appearing
- Score system included
---
## 🐍 2. Snake Game
- Snake moves using arrow keys
- Food appears randomly
- Score increases when food is eaten
- Level system increases speed
- Collision detection with walls
---
## 🎨 3. Paint Application
- Draw using arrow keys
- Change colors (red, blue, black)
- Draw shapes (circle, rectangle)
- Eraser mode included
